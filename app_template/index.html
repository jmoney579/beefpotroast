About this page

This page template was created by the Public Affairs Web Team for use by the Division of Information Systems for internal and publicly facing sites. The design conforms to established VDSS branding standards and policies. This template features:

  • Responsive behavior for use on mobile devices
  • Comments within the code describing it's purpose and providing structural guidance to encourage continued accessibility compliance
